===
Название: top
Описание: Интерактивный монитор процессов для Unix систем. Установлен по умолчанию (htop может не быть).
Синтаксис: top [опции]
Пример: top -o mem
Примечание: Показывает список процессов и их использование ресурсов, сортировка по используемой памяти с опцией -o mem

===
Название: man
Описание: Просмотр справки по командам Unix
Синтаксис: man [команда]
Пример: man grep
Примечание: Cправка по команде с описанием опций и примерами использования

===
Название: grep
Описание: Поиск строк по шаблону в файлах
Синтаксис: grep [опции] "шаблон" [файл]
Пример: grep "error" /var/log/system.log
Примечание: Фильтрации логов и вывод только нужной информации из файлов

===
Название: find
Описание: Поиск файлов и каталогов
Синтаксис: find [путь] [опции]
Пример: find /Users/username -name "*.txt"
Примечание: Ищет файлы с заданным именем или расширением в указанной директории

===
Название: head
Описание: Вывод первых строк файла
Синтаксис: head [опции] [файл]
Пример: head -n 10 /var/log/system.log
Примечание: Показывает первые строки файла(10 строк файла по умолчанию)

===
Название: tail
Описание: Вывод последних строк файла
Синтаксис: tail [опции] [файл]
Пример: tail -f /var/log/system.log
Примечание: Можно смотреть логи в реальном времени с опцией -f (допечатывает новые строки при их появлении в файле)

===
Название: wc
Описание: Подсчёт строк, слов и байт в файле
Синтаксис: wc [опции] [файл]
Пример: wc -l /var/log/system.log
Примечание: Опция -l показывает только количество строк (например записей логов)

===
Название: diff
Описание: Сравнение двух файлов построчно
Синтаксис: diff [опции] файл1 файл2
Пример: diff old.txt new.txt
Примечание: Выводит отличия между файлами (полезно при анализа изменений)

===
Название: uptime
Описание: Отображение времени работы системы и средней нагрузки
Синтаксис: uptime
Пример: uptime
Примечание: Показывает текущее время, сколько система работает и нагрузку за 1, 5 и 15 минут

===
Название: df
Описание: Информация об использовании дисков и файловых систем
Синтаксис: df [опции]
Пример: df -h
Примечание: Опция -h выводит размеры в человеко-читаемом виде (КБ, МБ, ГБ и тп)
